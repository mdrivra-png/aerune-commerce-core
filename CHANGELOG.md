# Changelog — Aerune Commerce Core V2

## V2 — Shared commerce build

- Added customer storefront, private Control Booth, and staff POS as one project.
- Added multi-storefront control: independent brands/websites with shared catalog and stock.
- Added collection-based product placement and product visibility per storefront.
- Added customer cart, checkout, online order desk, POS checkout, and stock deduction.
- Added cloud-ready Supabase schema with RLS, staff roles, image storage, real-time tables, and secure checkout RPC.
- Added product image upload, delete controls, drafts, publication status, low-stock controls, and CSV product export.
- Added individual SKU and barcode / scanner-code fields plus smart generators.
- Removed the hard-coded `BAG` behavior from newly created product codes.
- Added per-storefront hero image/video, custom content, custom domain field, and three visual modes.
